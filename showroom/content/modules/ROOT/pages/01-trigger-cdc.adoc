= Step 1: Trigger a CDC Event

== Action

Run this SQL INSERT in the terminal to create a new customer:

[source,bash,subs=attributes+]
----
PGPASSWORD=cdc-demo-2024 psql -h cdc-postgresql.kafka-cdc.svc -U cdcuser -d cdcdb -c \
"INSERT INTO customers (first_name, last_name, email) VALUES ('Demo', 'User', 'demo-test-$(date +%s)@neuralbank.io');"
----

TIP: A CronJob (`cdc-data-simulator`) is already inserting new customers every 60 seconds. You can also just wait and watch.

== Observable Event

. Open link:https://n8n-mailpit-openshift-lightspeed.{cluster_domain}[Mailpit^] — you should see a new email notification arrive within seconds
. The CDC pipeline automatically:
  * Captures the PostgreSQL WAL change (Debezium)
  * Publishes a CDC event to `cdc.public.customers` topic (Kafka)
  * Transforms and sends an email notification (HTTP Sink / Camel)
  * Triggers the SonataFlow onboarding workflow
