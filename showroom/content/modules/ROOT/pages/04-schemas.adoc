= Step 4: Schema Registry (Apicurio)

== Action

Open the link:https://apicurio-registry-kafka-cdc.{cluster_domain}[Apicurio Registry^] and browse the registered artifacts.

== Observable Event

You should see these registered schemas:

* `cdc.public.customers-value` — Debezium envelope schema for customer CDC events
* `cdc.public.orders-value` — Debezium envelope schema for order CDC events
* `bpm.approval.requests-value` — BPM approval request schema
* `bpm.onboarding.metrics-value` — Onboarding metrics schema

Each schema documents the exact structure of events flowing through the pipeline, enabling:

* Schema validation at produce/consume time
* Schema evolution with compatibility checks
* Contract-first API development
