= Step 6: DevSpaces IDE

== Action

Open link:https://backstage-developer-hub-developer-hub.{cluster_domain}[Developer Hub^] and navigate to *Create* → choose either:

* *Camel CDC Route (Kaoto)* — Visual Camel route editor
* *SonataFlow BPM Workflow* — Serverless Workflow visual editor

== Observable Event

A new DevSpaces workspace launches with pre-configured extensions:

* *Kaoto* — Visual low-code designer for Apache Camel routes
* *SonataFlow Editor* — Visual state machine editor for serverless workflows
* *Java/Quarkus* — Full IDE support for Camel/SonataFlow projects
* *YAML/XML* — Kubernetes manifest editing with validation

The workspace connects to an in-cluster Gitea repository where your project is scaffolded.
